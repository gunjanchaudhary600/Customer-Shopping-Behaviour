# Customer-Shopping-Behaviour
## End-to-End Data Analysis Project

## Project Overview
Is project me maine Excel, SQL, Python (Pandas), aur Power BI ka use karke ek complete end-to-end data analysis workflow perform kiya hai.
Iska main objective raw data ko clean karna, analyze karna, aur meaningful insights ke saath ek interactive dashboard banana tha.

## Dataset Summary
- Total Rows: 3900
- Total Columns: 18

## Key Features

# Customer Demographics
- Age, Gender, Location, Subscription Status, Purchase Details, Item Purchased, Category, Purchase Amount, Season, Size, Color

# Shopping Behavior
- Discount Applied, Promo Code Used, Previous Purchase, Frequency of Purchases, Review Rating, Shipping Type

# Missing Data
- 37 missing values found in the Review Rating column
- Missing values were handled during the data cleaning process

# Tools & Technologies Used
- Microsoft Excel – Data Cleaning
- SQL (PostgreSQL) – Data Extraction & Querying
- Python (Pandas) + Jupyter Notebook – Data Processing & Calculations
- Power BI – Interactive Dashboard Creation

## Project Workflow
1. Data Cleaning (Excel)
- Missing values handle ki
- Duplicate data remove kiya
- Data formatting & preprocessing kiya

2. Data Extraction (SQL)
- Database se required data extract kiya
- Queries likh kar filtered & structured dataset nikala
- Aggregations apply ki

3. Data Processing (Python - Pandas)
- Jupyter Notebook ko SQL se connect kiya
- Pandas ka use karke calculations perform ki
- Data transformation & analysis kiya

4. Data Visualization (Power BI)
- Processed data ko Power BI me import kiya
- Interactive dashboard create kiya
- KPIs & insights visualize kiye

## Key Highlights
- End-to-end data pipeline implementation
- SQL + Python integration
- Data cleaning to dashboard complete workflow
- Business-focused insights generation

Is project ke through maine data cleaning, querying, analysis aur visualization ka practical experience gain kiya.
